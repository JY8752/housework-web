# Housework-web

Next.js + Typescriptで家事管理アプリ。

## create-next-app

```CLI
npx create-next-app@latest --typescript
```

## Tailwind

```CLI
npm i -D tailwindcss@latest postcss@latest autoprefixer@latest
```

## Redux Toolkit

```CLI
npm install @reduxjs/toolkit
```

## firebase

```CLI
npm install firebase dotenv
```
